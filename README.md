# MTech Transport App Distribution

This repository hosts the public releases and updates for the **MTech Transport** Android application.

## 📲 Download Instructions

You can download the latest version of the app directly from this repository:

👉 **[Download Latest APK (v1.4)](https://raw.githubusercontent.com/EasterNM/mtech-app-dist/main/app-debug.apk)**

## 🛠️ Installation

1. Download the APK file on your Android device.
2. If prompted by your browser or system security, allow installation from unknown sources.
3. Open the downloaded file and install the application.
